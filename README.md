# LeKeylinks - WordPress 关键字自动链接插件

## 简介
LeKeylinks 是一个强大的 WordPress 插件，可以自动将文章内容中的指定关键字转换为相应的链接。它支持中英文关键字，并提供了灵活的配置选项，帮助您提升网站的内部链接建设。

## 功能特点
- 支持中文和英文关键字
- 可设置关键字替换优先级
- 可限制每篇文章中关键字的最大替换次数
- 智能识别已有链接，避免重复替换
- 防止在链接URL中的关键字被替换
- 支持批量管理关键字
- 简单易用的管理界面

## 安装方法
1. 下载插件压缩包
2. 登录 WordPress 后台，进入"插件">"安装插件"页面
3. 点击"上传插件"，选择下载的压缩包
4. 上传完成后点击"现在安装"
5. 安装完成后点击"启用插件"

## 使用说明

### 添加关键字
1. 在 WordPress 后台菜单中点击"关键字链接"
2. 点击"添加新关键字"按钮
3. 填写以下信息：
   - 关键字：需要被替换为链接的文字
   - 链接地址：关键字要链接到的目标URL
   - 优先级：数字越大优先级越高（当多个关键字匹配时优先使用优先级高的）
   - 最大替换次数：在单篇文章中最多替换的次数
   - 分词方式：选择"中文"或"英文"（英文会考虑单词边界）
4. 点击"保存关键字"按钮完成添加

### 管理关键字
- 在关键字列表页面可以查看所有已添加的关键字
- 可以对每个关键字进行编辑或删除操作
- 支持按关键字搜索
- 列表页面显示关键字的所有相关信息

### 注意事项
1. 关键字不要使用特殊字符，以免影响替换效果
2. 添加关键字时要注意避免使用 URL 中常见的字符串
3. 设置合理的最大替换次数，避免过度替换
4. 定期检查关键字的替换效果
5. 如果发现特定关键字导致问题，检查该关键字是否出现在其他链接的 URL 中

## 技术要求
- WordPress 版本：4.7 或更高
- PHP 版本：7.0 或更高
- MySQL 版本：5.6 或更高
- 需要启用 PHP DOM 扩展

## 常见问题

### 为什么某些关键字没有被替换？
- 检查关键字是否已达到最大替换次数
- 检查关键字是否出现在已有的链接中
- 检查关键字的优先级设置
- 确认关键字的分词方式是否正确

### 如何避免替换特定区域的关键字？
- 该关键字已经是链接的一部分时会自动跳过
- URL 中包含的关键字会自动跳过
- HTML 标签中的关键字会自动跳过

### 如何提高替换效率？
- 合理设置关键字优先级
- 避免添加过多相似的关键字
- 定期清理不需要的关键字
- 适当调整最大替换次数
